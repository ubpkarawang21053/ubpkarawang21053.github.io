Data Preparation Steps

  1. Pengumpulan Data:
    • Mengumpulkan data dari berbagai sumber, seperti basis data, file teks, web scraping, atau sensor.
    • Mengintegrasikan data dari berbagai sumber menjadi satu set data yang kohesif.
  2. Pembersihan Data (Data Cleaning):
    • Mengidentifikasi dan memperbaiki kesalahan dalam data, seperti nilai yang hilang, duplikasi, dan inkonsistensi.
    • Mengganti nilai yang hilang dengan nilai yang sesuai, seperti mean, median, atau menggunakan metode imputasi.
    • Menghapus data duplikat dan memastikan bahwa semua entri dalam dataset unik.
  3. Transformasi Data (Data Transformation):
    • Mengubah data menjadi format yang lebih cocok untuk analisis, seperti normalisasi atau standarisasi.
    • Menggabungkan atau memisahkan kolom data untuk membuat fitur baru yang relevan.
    • Mengubah tipe data sesuai dengan kebutuhan analisis, seperti mengonversi string menjadi numerik.
  4. Pengurangan Dimensi (Dimensionality Reduction):
    • Mengurangi jumlah fitur dalam dataset untuk menghilangkan fitur yang tidak relevan atau redundan.
    • Menggunakan teknik seperti Principal Component Analysis (PCA) atau feature selection untuk memilih fitur yang paling penting.
  5. Pembuatan Fitur (Feature Engineering):
    • Membuat fitur baru dari data yang ada yang dapat membantu meningkatkan kinerja model.
    • Menggunakan domain knowledge untuk mengekstrak informasi yang relevan dari data mentah.
  6. Pengkodean Kategori (Categorical Encoding):
    • Mengonversi data kategorikal menjadi format numerik yang dapat digunakan oleh algoritma machine learning.
    • Menggunakan teknik seperti one-hot encoding, label encoding, atau target encoding.
  7. Pembagian Data (Data Splitting):
    • Membagi dataset menjadi set pelatihan (training set) dan set pengujian (testing set) untuk memastikan model dapat diuji dengan data yang belum pernah dilihat sebelumnya.
    • Biasanya, data dibagi dengan proporsi seperti 70:30 atau 80:20.

Alat dan Teknik yang Digunakan dalam Persiapan Data

  1. Alat Perangkat Lunak:
    • Pandas (Python): Library yang kuat untuk manipulasi dan analisis data.
    • R: Bahasa pemrograman yang sering digunakan untuk statistik dan analisis data.
    • SQL: Bahasa untuk mengelola dan memanipulasi basis data relasional.
    • Excel: Alat yang sering digunakan untuk data kecil hingga menengah untuk pembersihan dan analisis data awal.
  2. Teknik Statistik dan Pemrograman:
    • Imputasi: Mengganti nilai yang hilang dengan mean, median, atau mode.
    • Normalisasi: Mengubah skala fitur sehingga berada dalam rentang yang sama.
    • Standarisasi: Mengubah fitur sehingga memiliki mean 0 dan standar deviasi 1.
    • Principal Component Analysis (PCA): Mengurangi dimensi data dengan menggabungkan fitur yang berkorelasi.
