BASIS DATA & DATA MINING

ditulis oleh Daniel Anju Saputra Simbolon

back

  1. Sample (Sampel):

    • Mengambil sampel data yang representatif dari kumpulan data yang lebih besar.
    • Memastikan sampel cukup besar untuk memberikan hasil yang valid tetapi cukup kecil untuk memungkinkan pemrosesan yang efisien.
    • Tujuan dari tahap ini adalah untuk memudahkan pengelolaan dan analisis data.

  2. Explore (Eksplorasi):

    • Menganalisis data secara deskriptif untuk menemukan pola, anomali, dan hubungan yang mungkin ada dalam data.
    • Menggunakan teknik visualisasi data dan statistik eksploratif untuk memahami karakteristik data.
    • Tujuan dari tahap ini adalah untuk mendapatkan wawasan awal tentang data dan mempersiapkan data untuk tahap berikutnya.

  3. Modify (Modifikasi):

    • Mengubah dan mempersiapkan data untuk pemodelan lebih lanjut.
    • Melakukan pembersihan data, transformasi data, dan pembuatan variabel baru yang relevan.
    • Mengatasi masalah data seperti nilai yang hilang, outlier, dan ketidaksesuaian.
    • Tujuan dari tahap ini adalah untuk memastikan data siap digunakan dalam pemodelan.

  4. Model (Pemodelan):

    • Membangun model data menggunakan teknik data mining seperti regresi, klasifikasi, clustering, atau asosiasi.
    • Memilih algoritma yang tepat dan mengoptimalkan parameter model.
    • Melakukan pelatihan model dengan data yang sudah dimodifikasi.
    • Tujuan dari tahap ini adalah untuk membuat model yang dapat memprediksi atau mengklasifikasikan data dengan akurasi tinggi.

  5. Assess (Penilaian):

    • Mengevaluasi kinerja model untuk memastikan model bekerja dengan baik pada data baru atau tidak terlihat.
    • Menggunakan metrik evaluasi seperti akurasi, presisi, recall, F-measure, dan ROC curve.
    • Membandingkan beberapa model untuk memilih model terbaik.
    • Tujuan dari tahap ini adalah untuk memastikan model yang dipilih memiliki kinerja yang memadai dan sesuai dengan tujuan bisnis.
